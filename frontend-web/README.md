# CollabPMS - Frontend

Frontend của hệ thống CollabPMS được xây dựng bằng **React + TypeScript + Vite**.

## Công nghệ
- React 19 + TypeScript
- Vite
- Material-UI (MUI) v5
- React Query
- dnd-kit (drag & drop)

## Hướng dẫn cài đặt

```bash
cd frontend-web
# Cài đặt dependencies
npm install

# Copy file môi trường
cp .env.example .env

# Chạy ở chế độ development
npm run dev
